# 🌩️ BeSA – AWS Foundations Series: Technical Challenges

> **Become a Solutions Architect** · Sample Files Repository

---

## 📖 About This Repository

Welcome to the official sample files repository for the **AWS Foundations Series**, part of the **BeSA (Become a Solutions Architect)** program.

This repository contains all the sample files, templates, configurations, and supporting assets used throughout the **Technical Challenges** of the AWS Foundations Series. These resources are designed to help participants follow along, practice hands-on skills, and successfully complete each challenge.

---

## 🎯 About the BeSA Program

The **BeSA – Become a Solutions Architect** program is a structured learning path designed to guide aspiring cloud professionals through the core competencies required to work as an AWS Solutions Architect.

The **AWS Foundations Series** is an introductory module within BeSA that covers fundamental AWS concepts and services, combining theoretical knowledge with practical, hands-on Technical Challenges.

## 🚀 Getting Started

### Prerequisites

Before using the files in this repository, make sure you have:

- An active **AWS SkillBuilder Account** 
- Basic familiarity with the **AWS Management Console**
- **Git** installed on your local machine

### Clone the Repository

```bash
git clone https://github.com/<your-org>/besa-aws-foundations.git
cd besa-aws-foundations
```

---

## 🛠️ How to Use This Repository

1. Navigate to the folder corresponding to your current **Technical Challenge**.
2. Read the challenge-specific `README.md` for instructions and context.
3. Use the provided sample files as directed by your BeSA instructor or the challenge guide.
4. Complete the challenge tasks in the SkillBuilder Lab

## ⚠️ Important Notes

- These files are intended **exclusively for educational purposes** within the BeSA program.
- Do **not** commit real AWS credentials, access keys, or sensitive data to this repository.
- Resources deployed to AWS during challenges may incur costs — always **clean up** your AWS resources after completing a challenge.
